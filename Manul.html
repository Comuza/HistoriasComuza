Introducción

Bienvenido al manual de Kerolmi. Aquí aprenderás los fundamentos del desarrollo web usando tecnologías puras.

Filosofía: Sin frameworks innecesarios, sin dependencias pesadas. Código limpio, comprensible y respetuoso con la privacidad.
Necesitas: Un editor de código como VS Code y un navegador web moderno.
Consejo: Enfócate en entender las bases antes de saltar a frameworks. ¡La maestría viene con la práctica y el conocimiento profundo!
Consejo 2: Que no te dé pena usar IA, no es malo usarla, solo no abuses de ella.
