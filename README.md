# ¡Kerolmi! • El Rincón de Comuza 

## Descripción
**¡Kerolmi!** es el proyecto fundacional que marca el inicio de mi trayectoria en el desarrollo web. Diseñado como un espacio personal y portafolio interactivo, destaca por una estética oscura, animaciones de rebote personalizadas y un enfoque directo en la experiencia del usuario a través de un diseño "underground".

## Propósito

**¡Kerolmi!** fue creado para centralizar mi presencia digital y mis primeros experimentos de código, enfocándose en tres pilares:
  
- **Identidad** • Un diseño único con colores `rgb(56, 1, 10)` que rompe con lo convencional. 
- **Conectividad** • Acceso directo a mi GitHub y LinkedIn para fomentar la colaboración.
- **Exploración** • Un sistema de navegación mediante un menú desplegable con físicas de rebote.

---

##  Características Originales (Legacy)

- **Estética Dark Original:** Uso de contrastes entre rojos profundos, azules medianoche y verdes oliva.
- **Animaciones de Autor:** Sistema de animaciones `@keyframes` como `tarjetaAnimada` y `rebote` programadas desde cero.
- **Menú Dinámico:** Botón "Menu" con efecto de pulso y despliegue con rotación de `-5deg`.
- **Sección Curricular:** Espacio dedicado para la visualización de experiencia y proyectos de GitHub.
- **Diseño Experimental:** Uso intensivo de sombras y transiciones para generar profundidad visual.
